# V3 页面清单（第一级广度需求）

> 接下来会逐个页面深入，定义每个页面的具体功能。

## 侧边栏

```
配置管理（可折叠）
  ├─ 字段管理
  ├─ 模板管理
  └─ NPC 管理

行为管理（可折叠）
  ├─ 事件类型
  ├─ 状态机
  └─ 行为树

世界管理（可折叠）
  └─ 区域管理

系统设置（可折叠）
  ├─ Schema 管理
  └─ 导出管理
```

## 9 个页面

| # | 页面 | 分类 | 一句话 |
|---|------|------|--------|
| 1 | 字段管理 | 配置管理 | 定义 NPC 可以有什么属性（血量、等级、性格...），是最底层原子单位 |
| 2 | 模板管理 | 配置管理 | 把字段组合成可复用的模板（战斗生物、环境NPC...） |
| 3 | NPC 管理 | 配置管理 | 选模板，填值，创建具体的 NPC 角色 |
| 4 | 事件类型 | 行为管理 | 定义游戏世界中会发生什么事（爆炸、枪声...） |
| 5 | 状态机 | 行为管理 | 定义 NPC 有哪些状态，什么条件下切换 |
| 6 | 行为树 | 行为管理 | 定义 NPC 在每个状态下具体做什么 |
| 7 | 区域管理 | 世界管理 | 定义游戏场景、天气、NPC 刷怪规则 |
| 8 | Schema 管理 | 系统设置 | 查看系统预定义的数据结构（开发者工具） |
| 9 | 导出管理 | 系统设置 | 导出配置给游戏服务端 |

## 推荐创建顺序

```
字段(1) → 模板(2) → 事件类型(4) → 状态机(5) → 行为树(6) → NPC(3) → 区域(7)
```
